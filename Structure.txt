Controllers/: Contains all the API controllers, each corresponding to a specific model.
Models/: Contains the model classes that represent the database tables.
Data/: Contains the DbContext class that serves as a bridge between your domain/entity classes and the database.
Migrations/: Contains the database migration files generated by Entity Framework Core. These files define how to create and update your database schema.
Properties/: Contains configuration files related to how the project is launched.
appsettings.json: Contains configuration settings for the application, including the database connection string.
appsettings.Development.json: Contains configuration settings specific to the development environment.
Program.cs: The entry point of the application where services are configured and the app is built.
TeamProjectAPI.csproj: The project file that contains information about the project, its dependencies, and how it should be built.
